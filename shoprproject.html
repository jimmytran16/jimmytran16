Shopr

Project Description

This is a Android mobile marketplace application. This application allows to you buy and sell your items within the platform. Includes a main page for user to log in or register if they haven't already. Also has a google sign-in feature to log into the platform using your pre-existing google account. Integrated the Google Firebase API for server-side configurations, such as user authentication & uploads and fetching items from the real time database.

Project Stack

  • Android Studios
  • Java
  • Google Firebase API
